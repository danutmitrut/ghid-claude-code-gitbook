# Partea I. Bază

**Setup și folosire productivă fără extensii. La final, lucrezi zilnic cu Claude Code ca agent în VS Code.**

---

## Ce conține

Cinci straturi care acoperă de la instalare până la folosirea zilnică completă:

[**1.1 Instalare**](instalare.md) setup de la zero pe macOS și Windows, cu Node.js, Git for Windows și Git Bash unde e cazul, plus extensia Claude Code și signin.

[**1.2 Primul contact**](primul-contact.md) elementele interfeței, primul prompt, primul diff, diferența față de Copilot și Cursor.

[**1.3 Integrare cu editorul**](integrare-editor.md) `@`-mentions pentru fișiere și linii specifice, shortcut `Option+K`/`Alt+K` pentru selecție, context automat din fișierul activ și terminal.

[**1.4 Workflow de editare**](workflow-editare.md) diff side-by-side, plan mode prin `/plan`, auto-accept cu `Shift+Tab`, modurile de permisiuni, conversation history.

[**1.5 Memorie de proiect**](memorie-proiect.md) CLAUDE.md la nivel de user și de proiect, ierarhia de încărcare, `/compact` și `/clear`, slash commands custom.

---

## Prerechizite pentru Partea I

- Cont Anthropic Pro, Max, Team, Enterprise sau Console
- Access administrator pe calculator pentru instalare
- Un proiect sau workspace pe care să exersezi după setup

VS Code și Node.js nu sunt prerechizite, sunt parte din stratul 1.1 Instalare.

---

## Ce ai după Partea I

- Ai Claude Code funcțional pe sistemul tău, indiferent că lucrezi pe Mac sau Windows
- Lucrezi productiv cu Claude Code în VS Code fără să explici context prin text
- Controlezi ce fișiere citește și ce schimbări aplică prin `@`-mentions și moduri de permisiuni
- Păstrezi context între sesiuni prin CLAUDE.md și conversation history
- Ai slash commands custom pentru task-urile tale repetitive

---

## Ordinea recomandată

Partea I se parcurge liniar. Fiecare strat presupune pe cel anterior. Dacă ai baze din alte IDE-uri AI (Copilot, Cursor), 1.2 Primul contact poate părea trivial, dar citește măcar capcanele, pentru că modelul mental e diferit.
